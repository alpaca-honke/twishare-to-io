# ブラウザ拡張Twishare to Misskeyにおけるプライバシーポリシー（個人情報保護方針）

あるかっぱ/アルパカ本家（以下、「制作者」という。）は、ブラウザ拡張機能Twishare to Misskey（以下、「本拡張機能」という。）ユーザーの個人情報について以下のとおりプライバシーポリシー（以下、「本ポリシー」という。）を定めます。本ポリシーは、本拡張機能や制作者がどのような個人情報などを取得し、どのように利用・共有するか、ユーザーがどのようにご自身の個人情報を管理できるかをご説明するものです。

## 1. 制作者情報

- 活動名：あるかっぱ/アルパカ本家 Alkappa/alpaca-honke  
- 連絡先：[こちら](https://alpaca-honke.github.io/)をご覧ください。  

## 2. 個人情報の取得方法

制作者は、当該拡張機能が取得する個人情報（個人情報の保護に関する法律第二条における定義に従ったもの）及びそれに準ずるものを取得することはありません。  
ただし、ChromeウェブストアやFirefox Browser Addonsなどの拡張機能配布ウェブサイトなどから取得できる統計情報は取得することができます。
これらの情報には、個人を特定できる情報は含まれません。  

## 3. 個人情報の利用目的

ブラウザ拡張内では、以下のように情報を取得し、利用します。  

- 設定画面に入力されたシェア先のサーバーのドメイン、その他設定情報：拡張機能及びブラウザにログインしているアカウント（Google ChromeであればGoogleアカウントなど）に保存され、Misskeyにシェアされる際などに使用されます。
- 表示中のウェブサイトのURL及び表示されているコンテンツ：Twitterにシェアされた内容を取得するため、ウェブページに独自のボタンなどを表示するため

拡張機能配布ウェブサイトなどから取得した情報は以下の目的で利用します。

- 拡張機能のインストール・利用状況などを分析し、今後の開発の参考にするため

また、制作者が取得した情報は、断りなく自身のMisskeyアカウントなどで公開する可能性があります。（個人を特定する情報は含まれません。）

## 4. 個人データを安全に管理するための措置

制作者は拡張機能内で使用する情報及び取得した情報を正確かつ最新の内容に保つよう努め、不正なアクセス・改ざん・漏えい・滅失及び毀損から保護するための努力を怠らないものとします。

## 5. 個人データの共同利用、第三者提供について

制作者は、上記項目3で言及した用途（取得した統計情報を自身のMisskeyアカウントなどで公開する）以外でのデータの共同利用、第三者提供を行いません。
また、法令及びガイドラインに別段の定めがある場合を除き、同意を得ないで第三者に個人情報を提供することは致しません。

## 6. 保有個人データの開示、訂正

制作者は拡張機能から個人に関する情報を取得しないため、個人データを保有しませんので、開示、訂正の要求にお応えすることはできません。
